# 📚 Caderno de Estudo — Probabilidade e Estatística (Teste 2)

Bem-vindo ao teu caderno de estudo, organizado **por temas** para estudares uma coisa de cada vez. Cobre exatamente o que sai no **Teste 2**: **Capítulo 4 (Variáveis Aleatórias Reais)** e **Capítulo 5 (Inferência Estatística)**.

## 🧭 Como usar este caderno

- Cada ficheiro segue a mesma estrutura: **explicação → definições → fórmulas (o que é / quando usar / como aplicar / que exercício pede) → exemplos passo a passo → erros comuns → ligação aos exercícios e ao teste de exemplo → resumo → perguntas de autoavaliação**.
- No fim de cada ficheiro, responde à **mini-lista de perguntas** — se não conseguires, relê essa secção.
- Usa o **[mapa de decisão](12_plano_estudo_90min.md#6-mapa-mental-de-decisão-que-ferramenta-usar)** (ficheiro 12) para treinar a identificar depressa que ferramenta usar.
- Os valores dos exemplos estão verificados contra as **soluções oficiais** dos PDFs em `content/`.
