Amostra de $n = 50$ produções de energia (kW/h), população normal. Em R:

**(a)** IC a 95% para a média;

**(b)** comente a afirmação "a variância é 12";

**(c)** teste se a proporção de geradores abaixo de 5 kW/h ($p$, defeituosos) é $\le 10\%$;

**(d)** há evidência de normalidade?

## Solução

```r
x <- c(15.01,3.81,2.74,16.82,14.30, ...)   # os 50 valores
t.test(x)$conf.int                          # (a)
n <- length(x); s2 <- var(x)
(n-1)*s2/12                                  # (b) qui-quadrado vs sigma^2 = 12
prop.test(sum(x < 5), n, p = 0.1, alternative = "greater")   # (c)
shapiro.test(x)                              # (d)
```

- **(a)** $IC_{95\%}(\mu) = \boxed{\,]8{,}935;\ 11{,}136[\,}$ (média $\approx 10{,}04$, via $t$).
- **(b)** o teste $\chi^2$ a $H_0: \sigma^2 = 12$ **não é rejeitado**: não há evidência de que a variância seja diferente de 12.
- **(c)** $H_0: p \le 0{,}1$ vs $H_1: p > 0{,}1$ — **não se rejeita**: não há evidência de que os defeituosos excedam 10%.
- **(d)** Shapiro-Wilk: **rejeita** a normalidade a 5% e 10%, mas **não** a rejeita a 1% (valor-p entre 0,01 e 0,05).
