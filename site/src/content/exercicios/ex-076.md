Seis porções de fertilizante deram percentagens de nitrogénio: $6{,}2\; 5{,}7\; 5{,}8\; 5{,}8\; 6{,}1\; 5{,}9$. O processo é normal com $\sigma^2 = 0{,}25$.

**(a)** Ao nível 10%, suportam os dados que $\mu = 6$?

**(b)** Responda pelo valor-p.

## Solução

$H_0: \mu = 6$ vs $H_1: \mu \ne 6$ (bilateral). Como $\sigma$ é **conhecido**, a estatística é $Z = \dfrac{\bar X - 6}{\sigma/\sqrt n} \overset{H_0}{\sim} N(0,1)$. Com $\bar x = 5{,}9167$, $\sigma = 0{,}5$, $n = 6$:

$$z = \frac{5{,}9167 - 6}{0{,}5/\sqrt{6}} = \frac{-0{,}0833}{0{,}2041} = -0{,}41.$$

**(b)** Valor-p bilateral: $\text{valor-p} = 2\,P(Z > 0{,}41) = 2(1 - 0{,}6591) = \boxed{0{,}6818}$.

**(a)** Como $\text{valor-p} = 0{,}6818 > 0{,}10 = \alpha$, **não se rejeita $H_0$**: os dados são compatíveis com $\mu = 6$.
