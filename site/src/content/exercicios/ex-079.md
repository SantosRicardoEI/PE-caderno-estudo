Na base `survey` (package `MASS`):

**(a)** teste se a pulsação média das mulheres é 73 e a dos homens 75;

**(b)** teste se a pulsação média dos fumadores é 85;

**(c)** crie a diferença entre o palmo da mão que escreve e o da outra e, na população feminina, teste se a primeira é maior.

## Solução

Todos são testes à média de população (aprox.) normal com variância desconhecida — `t.test` em R.

```r
library(MASS)
# (a) bilateral por sexo
t.test(Pulse ~ 1, data = subset(survey, Sex == "Female"), mu = 73)
t.test(Pulse ~ 1, data = subset(survey, Sex == "Male"),   mu = 75)
# (b)
t.test(subset(survey, Smoke != "Never")$Pulse, mu = 85)
# (c) diferença de palmos, unilateral (>0), só mulheres
dif <- with(subset(survey, Sex == "Female"), Wr.Hnd - NW.Hnd)
t.test(dif, alternative = "greater", mu = 0)
```

- **(a)** valores-p altos → **não se rejeita**: compatível com 73 (mulheres) e 75 (homens).
- **(b)** valor-p pequeno → **rejeita-se**: a pulsação média dos fumadores **não** é 85.
- **(c)** valor-p alto → **não se rejeita**: não há evidência de que o palmo da mão que escreve seja maior.
