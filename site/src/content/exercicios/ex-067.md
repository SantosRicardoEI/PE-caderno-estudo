Para $X \sim N(\mu, \sigma^2)$ com $\sigma^2$ conhecido, usando `z.test` (package `BSDA`) e simulações de $N(-2, 9)$, ilustre:

**(a)** aumentar o grau de confiança aumenta o erro;

**(b)** menor variância conhecida diminui o erro;

**(c)** para aumentar a confiança e diminuir o erro, aumenta-se $n$.

## Solução

Num IC para a média com $\sigma$ conhecido, o **erro** (semi-amplitude) é

$$\varepsilon = z_{1-\alpha/2}\cdot\frac{\sigma}{\sqrt{n}}.$$

Lendo a fórmula:

- **(a)** Mais confiança $\Rightarrow$ $z_{1-\alpha/2}$ **maior** $\Rightarrow$ $\varepsilon$ **maior** (o IC alarga).
- **(b)** $\sigma$ **menor** $\Rightarrow$ $\varepsilon$ **menor** (o IC estreita), mantendo $n$ e a confiança.
- **(c)** $\varepsilon$ desce com $\sqrt{n}$; aumentando $n$ pode-se **simultaneamente** subir a confiança e manter (ou baixar) o erro.

```r
library(BSDA)
set.seed(1)
x <- rnorm(30, mean = -2, sd = 3)      # sd = sqrt(9)
z.test(x, sigma.x = 3, conf.level = 0.90)$conf.int   # compara com 0.99
z.test(x, sigma.x = 1, conf.level = 0.90)$conf.int   # sd menor -> IC mais estreito
```

> 💡 Só se pode reduzir o erro **sem** perder confiança recolhendo **mais dados**.
