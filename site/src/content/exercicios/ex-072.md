Na base `iris`, para a espécie **virginica** (assumindo normalidade):

**(a)** ICs a 95% para o comprimento e a largura médios da sépala;

**(b)** implemente uma função R que devolva o IC para a variância de um $N(\mu, \sigma^2)$ e calcule-o para as duas variáveis.

## Solução

**(a)** IC da média com variância desconhecida ($t$) — em R, `t.test`:

```r
vir <- subset(iris, Species == "virginica")
t.test(vir$Sepal.Length, conf.level = 0.95)$conf.int   # ]6.4073, 6.7687[
t.test(vir$Sepal.Width,  conf.level = 0.95)$conf.int   # ]2.8823, 3.0657[
```

**(b)** O IC da variância usa os quantis do $\chi^2_{n-1}$: $\left]\dfrac{(n-1)s^2}{\chi^2_{1-\alpha/2}},\ \dfrac{(n-1)s^2}{\chi^2_{\alpha/2}}\right[$:

```r
ic_var <- function(dados, conf) {
  n <- length(dados); s2 <- var(dados); a <- 1 - conf
  c(inf = (n-1)*s2 / qchisq(1 - a/2, n-1),
    sup = (n-1)*s2 / qchisq(a/2,     n-1))
}
ic_var(vir$Sepal.Length, 0.95)   # ]0.2821, 0.6279[
ic_var(vir$Sepal.Width,  0.95)   # ]0.0726, 0.1615[
```

- **Resultados:** média do comprimento $]6{,}407;\ 6{,}769[$, média da largura $]2{,}882;\ 3{,}066[$; variância do comprimento $]0{,}282;\ 0{,}628[$, variância da largura $]0{,}073;\ 0{,}162[$.
